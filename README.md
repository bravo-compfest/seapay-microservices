# SEAPay Microservices

## Description
Microservices for SEAPay web application.

## Prerequisites
1. Java JDK >11 version

## Usage

### How to Build
1. Open directory of a microservice
2. Run this command below

        ./gradlew clean build

You can also run these microservices by using Docker (TBD).
